# Restaurant Management System – QA Project

## Overview

This project demonstrates my QA and Test Automation skills by testing a restaurant management system focused on reservations, table management, and user workflows.

## Testing Scope

* Functional Testing
* UI Automation Testing
* API Testing
* Regression Testing

## Tools & Technologies

* Cypress (UI Automation)
* Postman (API Testing)
* JavaScript
* Git & GitHub
              
## Key Features Tested

* Reservation creation and validation
* Table allocation logic
* Double booking prevention
* Edge cases and negative scenarios

## Project Structure

* `test-plan/` → Test strategy and scope
* `test-cases/` → Detailed test cases
* `bug-reports/` → Documented bugs
* `api-tests/` → Postman collections
* `automation/` → Automated test scripts

## Goal

To showcase real-world QA skills including test design, bug reporting, and automation.
